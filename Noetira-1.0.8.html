Noetira 1.0.8

  • 岗位本地初筛新增专业代码、规范化名称和专业层级判断，减少简单字符串匹配造成的误判。
  • 新增“已确认可报考”与撤销操作；需确认岗位可按需加入智能建议备选。
  • 智能建议改为目标岗位优先，并分为目标核心、需确认备选、已排除目标和总体组合四部分。
  • 移除岗位收藏及优先 200/500 范围，目标岗位筛选与顶部统计卡更醒目。
  • 公告检查改用低频中央采集生成的静态数据，减少客户电脑直接搜索导致的限流和封控。

公告能力仍处于测试阶段，建议从官方渠道自行下载并导入岗位表，核对原公告与招录机关答复。Noetira 完全退出后不能保证实时公告通知。

本次更新不会清除计划、专注、错题图片、复盘、阅读、岗位、激活信息或 API 配置。
